## 📌 Content Feed (similar to Facebook/Twitter Home Page)

### Poster

* As a user, I want to post photos of my catches so that I can share my achievements.
* As a user, I want to edit my posts so that I can correct or update the information I share.
* As a user, I want to delete my posts so that I can remove content I no longer want to share.
* As a user, I want to select a fish species when creating a post so that my catch is accurately categorized.
* As a user, I want to include the location of my catch in a post so that I can share where I fished with others.
* As a user, I want to add details (species, weight, bait, location) to my posts so that others can learn from it.

---

### Viewer / Commenter

* As a user, I want to find users and posts so that I can discover new content and people.
* As a user, I want to view discussions on fishing-related topics so that I can stay informed and learn from others.
* As a user, I want to view posts by category so that I can easily find relevant discussions.
* As a user, I want to like or upvote helpful posts so that high-quality content is highlighted to the community.
* As a user, I want to comment on posts so that I can interact with others and share insights about fishing locations.

---

## 🗺️ Map Page

* As a user, I want to discover recommended fishing spots so that I know where to start.
* As a user, I want to find fishing locations based on fish species so that I can target specific catches.
* As a user, I want to view the location associated with a post so that I can understand where the catch was made.
* As a user, I want to view posts near a location so that I can see recent activity in that area.

---

## 👤 User Profiles

* As a user, I want to create an account so that I can join the fishing community.
* As a user, I want to access my account so that I can view and manage my content.
* As a user, I want to manage my profile information so that others can learn about me.
* As a user, I want to add a profile image so that my account is more recognizable.
* As a user, I want to view other users' profiles so that I can see the content they have shared.
* As a user, I want to connect with other users so that I can build my network.
* As a user, I want to block other users so that I can control my interactions and avoid unwanted contact.

---

## 🔐 Login Page

* As a user, I want to create an account so that I can access the platform.
* As a user, I want to log into my account so that I can access my personal features.
